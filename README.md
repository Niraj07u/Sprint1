# Prodesk IT Landing Page

A modern, responsive, and professional landing page developed for **Prodesk IT** as part of the **Sprint 01 Deliverable**. This project demonstrates frontend development fundamentals using **HTML5**, **CSS3**, and **Vanilla JavaScript** while following responsive design principles, semantic HTML, and modern UI/UX practices.

---

##  Project Overview

The objective of this project was to design and develop a responsive landing page for a digital marketing and web solutions company. The application follows a clean and modern design with smooth user interactions, responsive layouts, and optimized frontend architecture.

The website includes multiple sections such as a Hero Banner, About, Services, Contact Form, and Footer while maintaining responsiveness across desktop, tablet, and mobile devices.
